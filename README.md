# Король POSAPI

Пиксельная браузерная игра про приём платежей на кассе.

Создана для презентации в детском саду — объясняет работу терминала оплаты в игровой форме.

## Играть

**[https://lllypuk.github.io/the-king-of-posapi/](https://lllypuk.github.io/the-king-of-posapi/)**

## Как играть

1. Нажми **"Начать смену"**
2. К кассе подходит покупатель с товарами
3. Посчитай сумму товаров в уме
4. Введи сумму на терминале и нажми **"Оплатить"** или **Enter**
5. Если верно — покупатель уходит, деньги зачисляются
6. Если неверно — попробуй ещё раз
7. Обслужи всех покупателей и заверши смену!

## Технологии

Один файл `index.html` — HTML + CSS + Canvas + Web Audio API. Никаких зависимостей.

## Запуск локально

Открой `index.html` в браузере.
