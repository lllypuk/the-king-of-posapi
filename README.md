# Король POSAPI

Пиксельная браузерная игра про приём платежей на кассе.

Создана для презентации в детском саду — объясняет работу терминала оплаты в игровой форме.

## Играть

**[https://lllypuk.github.io/the-king-of-posapi/](https://lllypuk.github.io/the-king-of-posapi/)**

## Как играть

1. Нажми **«Начать смену»**
2. К кассе подходит покупатель с товарами и ценами
3. Посчитай сумму товаров в уме
4. Введи сумму на терминале — нажимай цифры на экранной клавиатуре (или на обычной клавиатуре)
5. Нажми **✓** (или **Enter**) для оплаты
6. Если верно — терминал покажет анимацию обработки, покупатель уходит, деньги зачисляются
7. Если неверно — попробуй ещё раз
8. Обслужи всех 6 покупателей и заверши смену!

### Управление

| Действие | Экранная клавиатура | Клавиатура |
|----------|-------------------|------------|
| Ввод цифры | Кнопки 0–9 | Клавиши 0–9 |
| Оплатить | ✓ | Enter |
| Стереть | C | Backspace / Delete |
| Очистить | C | Escape / C |

## Технологии

Один файл `index.html` — HTML + CSS + Canvas + Web Audio API. Никаких зависимостей.

## Запуск локально

Открой `index.html` в браузере. Сервер не нужен.
