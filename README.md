# Король POSAPI

Пиксельные браузерные игры про профессии — для презентации в детском саду.

## Играть

**[https://lllypuk.github.io/the-king-of-posapi/](https://lllypuk.github.io/the-king-of-posapi/)**

## Игры

### Кассир («Король POSAPI»)
К кассе подходят покупатели с товарами. Посчитай сумму в уме, введи на терминале — обслужи всех 6 покупателей и заверши смену!

### Бэкенд-инженер
Ты следишь за серверами. Проблемы появляются в 4 зонах — успей кликнуть, пока сервис не упал. 3 жизни, чини пока можешь!

## Управление

### Кассир

| Действие | Экранная клавиатура | Клавиатура |
|----------|-------------------|------------|
| Ввод цифры | Кнопки 0–9 | Клавиши 0–9 |
| Оплатить | ✓ | Enter |
| Стереть | C | Backspace / Delete |
| Очистить | C | Escape / C |

### Бэкенд-инженер

| Действие | Мышь / тач | Клавиатура |
|----------|-----------|------------|
| Верхняя зона | Клик на проблему | 1 / W / ↑ |
| Левая зона | Клик на проблему | 2 / A / ← |
| Правая зона | Клик на проблему | 3 / D / → |
| Нижняя зона | Клик на проблему | 4 / S / ↓ |

## Технологии

Три HTML-файла — никаких зависимостей, никакого сервера.

- `index.html` — главное меню
- `cashier.html` — игра «Кассир»
- `backend.html` — игра «Бэкенд-инженер»

Canvas API + Web Audio API + CSS с пиксельным шрифтом Press Start 2P.

## Запуск локально

Открой `index.html` в браузере. Сервер не нужен.
