# Задачи разработки

## Этап 1: Каркас
- [ ] Создать `index.html` с базовой HTML-структурой
- [ ] Подключить пиксельный шрифт (Press Start 2P от Google Fonts или встроенный fallback)
- [ ] CSS-layout: верхнее табло, игровая зона, нижняя панель сообщений
- [ ] Настроить Canvas нужного размера

## Этап 2: Пиксельная графика
- [ ] Нарисовать фон: пол магазина, прилавок/кассу
- [ ] Нарисовать пиксельных покупателей (3-4 варианта внешности, разные цвета)
- [ ] Нарисовать товары (простые пиксельные иконки или использовать эмодзи)
- [ ] Нарисовать терминал оплаты

## Этап 3: Игровая логика
- [ ] Структура данных: массив покупателей, у каждого — массив товаров
- [ ] Генерация случайных покупателей при старте игры
- [ ] Отображение текущего покупателя и его товаров на прилавке
- [ ] Поле ввода суммы (стилизованное под терминал)
- [ ] Проверка введённой суммы: верно / неверно
- [ ] Счётчик заработанных денег
- [ ] Переход к следующему покупателю после успешной оплаты

## Этап 4: Анимации и переходы
- [ ] Анимация: покупатель подходит к кассе (сдвиг спрайта)
- [ ] Анимация: покупатель уходит после оплаты
- [ ] Анимация: появление товаров на прилавке
- [ ] Визуальный фидбек: зелёная вспышка при успехе, красная при ошибке

## Этап 5: Звуки (опционально)
- [ ] Звук "бип" при нажатии кнопки оплаты
- [ ] Звук успешной оплаты (весёлый)
- [ ] Звук ошибки (мягкий, не пугающий)

## Этап 6: Экраны и полировка
- [ ] Стартовый экран с названием и кнопкой "Начать смену"
- [ ] Экран победы: "Смена окончена!", итоговая сумма, кнопка "Играть ещё"
- [ ] Адаптация под разные размеры экрана (чтобы работало на проекторе)
- [ ] Финальное тестирование и подгонка баланса (количество покупателей, сложность сумм)
