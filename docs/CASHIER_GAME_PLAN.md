# Король ПОС-API — Пиксельная игра про приём платежей

## Цель

Простая пиксельная игра для презентации в детском саду.
Объясняет работу терминала оплаты в игровой форме.

## Концепция

Игрок — кассир. К кассе подходят покупатели с товарами.
Нужно посчитать сумму покупок и правильно ввести её на терминале оплаты.

## Технологии

- **Один файл `index.html`** — HTML + CSS + JavaScript
- **Canvas API** — пиксельная графика
- **Web Audio API** — простые звуки (бип при оплате)
- **Никаких зависимостей** — открыл в браузере и играй

## Экраны

1. **Стартовый экран** — название игры, кнопка "Начать смену"
2. **Игровой экран** — основной геймплей
3. **Экран победы** — "Смена окончена! Все покупатели обслужены!"

## Игровой экран — расположение элементов

```
┌──────────────────────────────────────────────┐
│          💰 Заработано: 0 ₽                  │
├──────────────────────────────────────────────┤
│                                              │
│  🧑🧑🧑  →  [ Касса / Прилавок ]  [ Терминал ] │
│  очередь      товары + цены       ввод суммы │
│                                              │
├──────────────────────────────────────────────┤
│  Сообщение: "Введите сумму на терминале!"    │
└──────────────────────────────────────────────┘
```

## Механика

1. Покупатель подходит к кассе
2. На прилавке появляются его товары с ценами (например: 🍎 5₽, 🥛 10₽, 🍞 8₽)
3. Игрок считает сумму в уме и вводит на терминале
4. Нажимает "Оплатить" (или Enter)
5. **Верно** → зелёная галочка ✅, звук успеха, сумма добавляется к заработку, покупатель уходит
6. **Неверно** → красный крестик ❌, сообщение "Попробуй ещё раз!", можно ввести заново
7. После всех покупателей — экран победы

## Покупатели и товары

6 покупателей за «смену». У каждого 2–4 товара.

| Товар     | Цена  |
|-----------|-------|
| Яблоко 🍎 | 2 ₽   |
| Молоко 🥛 | 5 ₽   |
| Хлеб 🍞   | 3 ₽   |
| Сок 🧃    | 4 ₽   |
| Конфета 🍬 | 1 ₽   |
| Печенье 🍪 | 3 ₽   |
| Банан 🍌   | 2 ₽   |
